# Atividades do Capítulo 2 - Integração com HTML

Neste capítulo, pratiquei a integração de JavaScript com HTML e algumas funcionalidades

## Conteúdo principal

- Estrutura básica de HTML
- Manipulação de elementos com `getElementById()`
- Propriedades `textContent`, `innerHTML` e `value`
- Eventos e funções em JavaScript
- Variáveis, operadores e funções matemáticas
- Exercícios práticos

## Objetivo

Aplicar conceitos para desenvolver habilidades em interatividade e manipulação do DOM usando JavaScript.
